AiFinPay is multichain by design — chains are execution environments, not products. The same SDK and the same fee-on-top model work across every network. The only thing that changes per chain is how an agent proves it paid.

AiFinPay is token-free: there is no protocol token to buy, stake, or hold. Agents pay per request in the native asset or a stablecoin, and a flat protocol fee is added on top and settled atomically.

## Fee model

**Fee-on-top, atomic split.** A flat protocol fee is added on top of the provider's price and settled in one transaction — 99% provider / 1% treasury / 0.01% creator. No token, no staking; pay per request. Treasury is multisig-owned (Gnosis Safe / Squads).

## How an agent proves payment

Settlement proof is per-chain. The SDK handles the right scheme automatically for the network you target.
